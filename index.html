Making the Switch

You'll need the following...

PHP Code Sniffer

Install PHPCS on your machine using PEAR or Composer. Alternatively, use Homebrew if you are an OSX user.

Add the path to your phpcs command to your system PATH, transforming it into a global installation.

WPCS from GitHub

Download the WPCS rules for PHPCS to your machine using Git, Pear, Composer or the old fashioned way.

Place them in suitable directory and make a note of the location. Mine are in: /Users/davetgreen/.wpcs

A Code Editor / IDE

Sublime
Atom
Netbeans
PHPStorm
Coda

Or any decent code editor / IDE that has either a plugin ecosystem or native support for PHP Code Sniffer.

EDITOR SUPPORT

Install and configure the relevant plugin for your code editor / IDE, or turn on and configure native support.

You will need to provide the path to the directory containing the WPCS rules you downloaded from GitHub.
